Lime-form
 
 * Dynamic Form Builder form generating plugin for rapid development and deployment 
  
 
 Features:
 -Sends Email
 -File Upload 
 -Ajax enabled
 
 * Uses Valitron validation ( https://github.com/vlucas/valitron )
 * Uses PHPMailer for emails ( https://github.com/PHPMailer/PHPMailer)
